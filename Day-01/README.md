# Linux Practice - Day 1

## 📂 Task Performed

- Created website folder
- Created css, js, and images directories
- Created project files
- Copied files
- Renamed files
- Deleted files and folders
- Viewed final structure

---

## 💻 Commands Used

```bash
mkdir website
cd website

mkdir css js images

touch index.html style.css app.js

cp style.css backup_style.css

mv app.js main.js

rm backup_style.css

rmdir images

ls -l
```

---

## 📘 What I Learned

- Creating directories using `mkdir`
- Navigating folders using `cd`
- Creating files using `touch`
- Copying files using `cp`
- Renaming files using `mv`
- Deleting files using `rm`
- Removing empty directories using `rmdir`
- Viewing files and folders using `ls -l`
