# 🍔 Food Delivery System — Linux Practice

## 📌 Scenario

Worked on a Linux Support Engineer practice scenario for managing an online food delivery server system.

The goal of this practice was to improve Linux command-line skills through real-world file and directory management tasks.

---

## 🛠 Tasks Performed

* Created project directories
* Managed order and customer files
* Added logs using `echo`
* Read file contents using `cat`
* Created backups
* Renamed and moved files
* Worked with hidden files
* Removed unnecessary data

---

## 📂 Project Structure

```bash
food-delivery-system/
├── orders/
├── customers/
├── logs/
└── backup/
```

## 📚 What I Learned

* Linux file and directory management
* Writing and reading file data
* Backup handling
* Hidden file management
* Linux project organization
* Basic Linux administration tasks
