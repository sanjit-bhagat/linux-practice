# 🏥 Hospital Data Management System (Linux Practice)

## 📌 Overview  
Practiced a Linux scenario to manage hospital data with proper user roles and permissions.

## 💻 Tasks
- Created users & groups  
- Managed ownership (`chown`)  
- Set permissions (`chmod`)  
- Secured patient data  
- Performed backups  

## 🔐 Access Control
- Doctors → Read/Write patients  
- Reception → Billing access  
- Admin → Full access  
- Others → No access  

## 💡 Learnings
- Real use of `660`, `750`, `770`  
- Importance of data security  
- Practical Linux system management
